Arguments for the validate method

## Hierarchy

* **IValidationArgs**

## Properties

###  checkAsync

• **checkAsync**: *boolean*

True, if the async validators should
be triggered as well, otherwise only
the sync validators are triggered

Default: true

___

###  immediateAsync

• **immediateAsync**: *boolean*

True, if the async validators should
be triggered without any delay

Default: false
